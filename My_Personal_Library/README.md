# AbdelFattah Library 📚

A beautifully crafted, responsive personal digital library interface designed to showcase selected books across three core disciplines: Arabic Grammar & Philology, Self-Improvement, and Software Engineering. 

This project was built from scratch to practice clean semantic HTML5 and vanilla CSS3 styling principles, focusing on precise structural layout and harmonious color palettes.

---

## 🚀 Features

*   **Semantic Structure:** Built using modern HTML5 layout tags (`<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<figure>`).
*   **Custom Typography:** Integrated elegant Arabic and English typography via Google Fonts (featuring *Reem Kufi Fun* and *Marhey*).
*   **Harmonious Color Palette:** Designed with a premium dark navigation bar (`#1a1e21`) matched with traditional book-paper tones (`#f4f1ea`) for the article cards.
*   **Clean CSS Layout:** Structured utilizing the standard Box Model, precise floats, dynamic cross-browser logic properties (`margin-inline-start`), and smooth transition hover effects.
*   **Fully Clean Code:** Avoided codebase bloat and unnecessary styling redundancies for optimal rendering performance.

---

## 🛠️ Built With

*   **HTML5** - Structuring the web content.
*   **CSS3** - Custom styling, gradients (`radial-gradient`), and layout management.
*   **Google Fonts** - Web typography integration.
