# Facebook Sign-Up Page Clone 🚀

A responsive replica of the Facebook mobile sign-up interface, built as a practical milestone to master semantic HTML and structured layouts.

## 🛠️ Tech Stack
- **HTML5:** Structured using forms, labels, input types (`tel`, `password`), and customized select groups.
- **CSS3:** Styled with custom borders, rounded corners, centered containers, and clean typography.

## 💡 Project Insights & Architecture
This project represents a crucial phase in my development journey. Instead of jumping directly into advanced CSS layouts like Flexbox, I purposefully utilized standard structured layouts and HTML tables to group related components (like the Date of Birth dropdowns). 

As a developer, I believe true problem-solving comes from maximizing current knowledge before scaling up. This codebase serves as a solid baseline that I will refactor and optimize once I advance into CSS Flexbox and responsive box-modeling.
