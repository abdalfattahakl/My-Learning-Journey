# 🌐 بوابة المعلم الذكية | Smart Teacher Portal

بوابة إلكترونية تعريفية تعليمية متكاملة، تم بناؤها وتطويرها بالكامل باستخدام **HTML5** و **CSS3**. تهدف البوابة إلى دمج عراقة لغتنا العربية والمنظومة التربوية بذكاء التقنية وتطوير واجهات الويب الحديثة.

---

## 📄 هيكلية المشروع ووصف الصفحات

الموقع يتكون من **4 صفحات رئيسية مترابطة** بشكل شبكي مرن ومستقر:

1. **الصفحة الرئيسية (`index.html`):** معرض الأعمال الشخصي والنبذة التعريفية للمطور والتربوي، مع استخدام عناصر هيكلية (Semantic Elements) متوافقة مع محركات البحث.
2. **مكتبة التحميلات (`downloads.html`):** جدول رقمي متقدم ومصمم بذكاء باستخدام دمج الصفوف المتقدمة (`rowspan`) لإتاحة تحميل أمهات كتب القواعد النحوية (مثل النحو الواضح وجامع الدروس) بصيغ رقمية آمنة.
3. **الفصل الافتراضي (`classroom.html`):** فضاء تعليمي مرئي يعتمد على التضمين المرن للفيديوهات الرقمية (`<iframe>`) باستغلال الأبعاد النسبية المئوية لضمان تناسق العرض.
4. **اتصل بنا (`contact.html`):** جدول تواصل حي يربط الزائر مباشرة ببروتوكولات الاتصال الفوري (الهاتف والبريد الإلكتروني والشبكات الاجتماعية).

---

## 🛠️ التقنيات والمفاهيم الهندسية المستخدمة

* **هيكلة نظيفة (Clean HTML5 Semantic Structure):** الالتزام الكامل بوسوم مثل `<header>`, `<nav>`, `<main>`, `<section>`, `<footer>`.
* **التحكم بالصناديق (Box Model & Layout):** فهم وتطبيق فلسفة التموضع والتوسيط من خلال `margin: auto` للعناصر الكتليّة والـ `text-align: center` للعناصر المضمنة.
* **تحويل السلوك (Display Manipulation):** تحويل سلوك القوائم والـ Blocks الجشعة إلى صفوف أفقية مرنة باستخدام `display: inline-block`.
* **إدارة الموارد المستدامة:** تنظيم المطبخ البرمجي بفصل الملفات ومجلد الاختصار الذكي للصور والموارد `img/` و `files/`.
